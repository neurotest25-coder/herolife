## Что за проект
HeroLife — веб-приложение геймификации привычек
Целевая аудитория: женщины России
Хранение: localStorage (сервер в будущем)
Будет конвертироваться в мобильное приложение
Все элементы делать mobile-first (минимум 44px кнопки)

## Стек
- HTML + CSS + JS (vanilla, без фреймворков)
- Файлы: index.html, css/style.css, js/data.js, 
  js/pet.js, js/inspiration.js, js/shop.js, 
  js/stats.js, js/app.js
- localStorage для данных
- Git для версий
- Позже: Flask + сервер

## Важно для Cursor
- Писать промпты на русском языке
- Один промпт = одна задача = один файл
- Всегда писать "остальное не трогай"
- После каждого промпта проверять в браузере
- Глобальные функции экспортируются через window.*
- Порядок подключения JS строго:
  data.js → pet.js → inspiration.js → 
  shop.js → stats.js → app.js

## Как P передаётся между файлами
- P живёт внутри app.js в замыкании
- Другие файлы получают через window.getP()
- Другие файлы меняют через window.setP(P)
- После изменений вызывать window.save() и window.render()

## Концепция (важно!)
Тон = мудрая подруга
- НЕ коуч, НЕ учитель, НЕ мама
- Поддерживает, не оценивает
- Мягкий streak (не наказывает за пропуск)
